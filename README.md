# Shoduino Line Follower 🏁🏎💨

## Library Installation 📚

Installing the ***Shoduino Line Follower*** library is a straightforward process, but getting the folders in the right spot is key for the Arduino IDE to recognize the custom hardware.

### Prerequisites 💻
Before starting, ensure you have the Arduino IDE installed. If you don't have it yet, you can find the latest version at https://www.arduino.cc/en/software/

## Installation Steps 📌

1. **Download the Repository**

   Visit https://github.com/Uus3r199/project_shoduino-v1. Click the green ***Code*** button and select ***Download ZIP***.

2. **Extract the File**

   Once the download is complete, extract the contents to a location you can easily access (like your Desktop or Downloads folder).
   
3. **Open the Extracted Folder**

   Open the newly extracted folder `project_shoduino-v1-main`. Inside, you will see a folder named `project_shoduino v1`

4. **Move the project_shoduino v1 folder**

   Take that `project_shoduino v1` folder and move it to `Documents → Arduino → *here*`

5. **Open Arduino IDE**

   Open the Arduino IDE, go to the ***Sketchbook***, and you should be able to see `project_shoduino v1` there, just as shown in the image.
   
   <img width="718" height="361" alt="002abc66-9d97-413d-8cda-f0ffe2500e5d" src="https://github.com/user-attachments/assets/386c6b3c-f7da-41c5-9f22-60b025a20d3e" />
  
7. **Add ZIP Library**

   In the Arduino IDE, go to `Sketch → Include Library → Add .ZIP Library`. Choose the `project_shoduino v1.zip` inside the **project_shoduino-v1-main** folder

   <img width="617" height="343" alt="aa8141dd-e44d-4cae-8f81-06bc37a0a31c" src="https://github.com/user-attachments/assets/d97549d2-da32-442d-a866-bbd66c6e816a" />
   
9. **Compile/Upload**
 
   Open `main_sketch` in the Sketchbook and click Verify/Compile. Upload and start racing.
